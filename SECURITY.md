# Security Policy

## Supported scope

This repository is public for product transparency and engineering collaboration. Production secrets, customer data, and deployment credentials are intentionally excluded.

## Reporting a vulnerability

If you identify a security issue, please report it privately to:

- `starkaid24@gmail.com`

Please include:

- a concise description of the issue
- reproduction steps
- impacted area or endpoint
- suggested mitigation, if available

## Sensitive data handling

Never open a public issue containing:

- live API keys
- database credentials
- webhook secrets
- customer records
- payment metadata tied to real users

Use private contact first for anything with real impact.
